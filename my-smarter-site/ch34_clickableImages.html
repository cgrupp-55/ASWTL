Adding More Info to the Image Tag

The minimal amount of imnformtion necessary to define an image tag is img src="path/fileName"

However, additional information can be inserted to provide text as an alternative to display which may be presented in place of the image in case the browser fails to display the image or when a user is using a screen reader to view the content.

The format to diplay alternative text within the img src tag is alt="text".

In addition, the display width and height of the image can be modified.

The format is width="size" and height="size" where size is the number of pixels.

Note: browsers don't care which order you specify src, alt, width and height

Also note: image elements can be converted from inline to block elements by including in the related css files "display: block;" classes with the format in the html file of ClassName="Block" command included in the img src tag.



Note: To allow users to resize the text (in the browser menu), many developers use em instead of pixels.1em is equal to the current font size. The default text size in browsers is 16px. So, the default size of 1em is 16px.
