Positioning an Image

Browsers will always place an image to the far left of the screen unless directed otherwise.

The css command to set a margin for an image or text is the margin property.

The margin property options include:

(Set all margins set with the same spacing) margin: 35px;

(Set top, right, bottom, left margin properties) margin: 10px 5px 15px;

(Set left margin properties) margin-left: 2em;

(Set right margins properties) margin-right: 2em;

(Set top margins properties) margin-top: 2em;

(Set bottom margin properties) margin-bottom: 2em;

(Set top, both sides, and bottom margin properties) margin: 10px 5px 15px;

(Set vertical and horizontal margines with two values in margin properties) margin: 10px 5px;

Here the image elements are converted from inline to block elements by including in the related css files "display: block;" classes with the format in the html file of ClassName="Block" command included in the img src tag.
