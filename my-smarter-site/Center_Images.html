Centering an Image

Centering an image requires the addition of a css class to create the image centering instructions, plus adding the class into the img.src tags within the html families.

In the css file, add an img.className element containing display:block, margin-left: auto, and margin-right: auto properties.

Notice the Display: Block property removed the inline relationship between the above three images.
In addition, I added the padding: 1em; property to the css img class to create vertical spacing between iamge

Here the image elements are converted from inline to block elements by including in the related css files "display: block;" classes with the format in the html file of ClassName="Block" command included in the img src tag.
