Wrapping text around an Image with Float Properties

Wrapping text around an image requires the addition of a css class to create the image wrapping instructions, plus adding the class into the img.src tags within the html families.

In the css file, add an img.className, such as img.wrapLeft or img.wrapRight

The img.wrapRight class tells the html file that any text that follws this image with this class will wrap around the image on the left.

The img.wrapLeft class tells the html file that any text that follws this image with this class will wrap around the image on the right.

A p.noWrap class tells the html file not to wrap anything from this point forward in the page.
