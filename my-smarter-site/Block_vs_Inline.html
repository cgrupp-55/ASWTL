Block versus Inline HTML Elements

Html elements such as headings, paragraphs, lists, tables an divs are Block elements.

Block elements stand alone oin a web page. As you create each one, your browser will automatically place them on a new line.

You must use the float command to overcome the default of placing block elements in separate lines. This is done all the time with div elements to allow placement of sidebars and navigation buttons or tabs alongside the main text area.

All block elements inside a div element have their own horizontal space (line) inside the div.

The browser also adds spacing between each line on block elements.


Inline elements don't start on a new line.

Inline elements, such as the href element become part of the flow within a header or sentence, and therefore do not start a new line.
