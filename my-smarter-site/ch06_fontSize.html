In this example, I have changed the font sizes to all header and paragraph elements by creating classes in the style_6.css files specifying font-size: 1.5em.

1.5em is telling the browser to make the text in the headers and Paragraphs to be 1.5 times (100%) the default text size.

1em is the same size (100%) of the default text size.

.75em is three-quarters (75%) of the default text size.

2em is twice the size (200%) of the defaut text size.
